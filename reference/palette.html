Palette#

The GNOME color palette is intended for use in app icons and illustrations. A reference table is provided below.

The palette can also be accessed using the Palette app, through the predefined palettes in recent versions of the GIMP and Inkscape, or by downloading it in GIMP/Inkscape format.
